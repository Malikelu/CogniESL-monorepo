## 4. AI Optimization (AIO / SGE)

Per Google's AI optimization guide:

### Key Principles
1. **Create non-commodity content** — L1 interference data is unique, not available elsewhere
2. **People-first content** — Written for teachers, not search engines
3. **Expertise signals** — Demonstrate deep ESL knowledge
4. **Clear technical structure** — Proper heading hierarchy, semantic HTML

### AI Overview Optimization
- FAQ schema for direct question answers
- Clear, concise answers that AI can extract
- Data tables (error patterns, comparisons)
- Structured lists (steps, features)
- Authoritativeness through depth

### Content That AI Loves
- "Why do [language] speakers make [mistake]?" — direct question format
- Comparison tables (error → correction → explanation)
- Numbered lists (steps, tips, patterns)
- Clear definitions with examples
